Sobre o projeto:
- Um aplicativo que vai te ajudar a lidar com as suas relações, ou com o término delas, de maneira bem humorada.

Tecnologias utilizadas:

- React.js
- TypeScript
- Vite
- HTML5
- CSS3

1. Clone o repositório

4. Instale as dependências:
